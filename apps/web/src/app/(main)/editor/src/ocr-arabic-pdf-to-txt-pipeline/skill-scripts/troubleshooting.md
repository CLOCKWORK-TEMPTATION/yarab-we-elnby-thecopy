# حل المشاكل والحالات الحدية — Troubleshooting Guide

---

## 1. PDF محمي بكلمة مرور

**التشخيص:** سكريبت `classify-pdf.ts` يُرجع `type: "protected"`

**الإجراء:** أبلغ المستخدم فوراً — لا تحاول فك التشفير.

---

## 2. جودة المسح رديئة (DPI منخفض، ضوضاء، ميل)

**التشخيص:** نتائج OCR مليئة بـ `{غير واضح}` أو أحرف عشوائية

**الإجراء:**

1. شغّل `scripts/enhance-image.ts` على الصور قبل إعادة OCR
2. إذا لم تتحسن النتائج — أبلغ المستخدم أن جودة المسح غير كافية

**ملاحظة:** Mistral OCR 3 مُحسَّن أصلاً للتعامل مع:

- ضغط الصور (compression artifacts)
- الميل (skew) والتشوه (distortion)
- DPI منخفض
- الضوضاء والخلفيات الملوّنة

لذلك جرّب Mistral أولاً قبل تحسين الصور.

---

## 3. فشل Mistral API

**أسباب محتملة:**

- مفتاح API غير صالح أو منتهي
- الملف أكبر من 50MB (الحد الأقصى)
- أكثر من 1000 صفحة (الحد الأقصى لطلب واحد)
- Rate limit

**الإجراء:**

- مفتاح غير صالح → اسأل المستخدم عن المفتاح الصحيح
- ملف كبير → قسّم الملف بـ `pdfseparate` أو `qpdf`
- صفحات كثيرة → عالج على دفعات (500 صفحة لكل طلب)
- Rate limit → انتظر 60 ثانية ثم أعد المحاولة
- إذا فشل كل ما سبق → انتقل لـ DeepSeek-OCR أو Claude Vision

---

## 4. نص عربي مختلط مع لغات أخرى

**المشكلة:** أجزاء إنجليزية أو فرنسية تظهر مشوّهة

**الإجراء:** Mistral OCR 3 يدعم أصلاً النصوص متعددة اللغات.
إذا ظهرت مشاكل مع Claude Vision — التزم بقاعدة "أبقِ النص الأجنبي كما هو" ولا تحاول ترجمته.

---

## 5. الخطوط غير المعتادة (ديواني، نسخ مزخرف، رقعة يدوي)

**المشكلة:** الخطوط العربية المزخرفة صعبة على معظم محركات OCR

**الإجراء:**

1. Mistral OCR 3 → يدعم الخط اليدوي والمزخرف (أفضل خيار)
2. إذا فشل → جرّب Claude Vision مع تعليمات صريحة عن نوع الخط
3. إذا فشل الاثنان → أبلغ المستخدم وضع `{غير واضح}` في الأجزاء المشكلة

---

## 6. جداول معقدة (خلايا مدمجة، جداول متداخلة)

**الإجراء:**

- Mistral OCR 3: يُخرج HTML tables مع `colspan`/`rowspan` — الأدق
- إذا كانت الجداول مهمة جداً → اسأل المستخدم إذا يريد مخرج Markdown أو أنه يحتاج دقة أعلى (HTML)

---

## 7. المستند يحتوي صفحات فارغة أو صفحات صور فقط

**الإجراء:**

- صفحات فارغة: ضع `[صفحة فارغة]` في الموضع المناسب
- صفحات صور فقط: ضع `[صفحة تحتوي صورة فقط]`

---

## 8. التحقق المتقاطع بين المحركات

عند الشك في دقة النتائج، يمكن:

1. معالجة نفس الصفحة بمحركين مختلفين
2. مقارنة النتائج — التطابق يعني دقة عالية
3. الاختلافات → راجع يدوياً بالعودة للصورة الأصلية

**هذا المسار مكلف زمنياً — استخدمه فقط عند الحاجة الفعلية.**
